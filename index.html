Conexao com .exe

Passo a passo para autenticar seu executavel.

Fluxo recomendado

  1. Crie seu app em /app e pegue name, ownerid e secret.
  2. No seu .exe, chame o endpoint da API KeyAuth com a key do usuario.
  3. Se a resposta for valida, libera o programa; se falhar, bloqueia acesso.
  4. Use HWID lock para reduzir compartilhamento indevido de licenca.

Exemplo rapido de requisicao

POST /api/1.2/
Content-Type: application/x-www-form-urlencoded

&type=license
&name=SEU_APP
&ownerid=SEU_OWNERID
&secret=SEU_SECRET
&key=KEY_DO_USUARIO
&sessionid=... (apos init)
&init_iv=... (dependendo do SDK)

Dica: usar os SDKs oficiais da KeyAuth acelera a integracao no seu .exe.
